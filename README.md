# csn-da23ttb-tranthiyenkhoa-webtrangsuc-bootstrap
Đồ án cơ sở ngành: Phát triển website bán trang sức với dữ liệu từ API JSON.
## Giới thiệu đề tài
Đồ án này để xây dựng một website tĩnh bán trang sức, dùng html,css(bootstrap5), javascript
Dữ liệu sản phẩm được lấy từ API JSON thông qua JSON sever.
Các chức năng cơ bản của trang web:
<ul>
    <li>Trang chủ</li>
    <li>Trang danh mục (lọc sản phẩm)</li>
    <li>Trang chi tiết sản phẩm</li>
    <li>Tìm kiếm</li>
    <li>Giỏ hàng</li>
</ul>
## Công nghệ sử dụng:
<ul>
    <li>Frontend: HTML, CSS,Bootstrap 5, JavaScript</li>
    <li>API giả lập, JSON sever</li>
    <li>Cài JSON: npm install -g json-server</li>
    <li>Khởi động JSON: json-server --watch tenfilen.json</li>
    <li>Editor: VS Code</li>
    <li>Quản lý mã nguồn: Github</li>
</ul>
